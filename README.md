# KCModelEditor

[![AI Policy](https://img.shields.io/badge/ai_policy-human_contributions_only-green.svg)](https://codeberg.org/keepers-compound/kc-model-editor#ai-policy)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://codeberg.org/keepers-compound/kc-model-editor#license)

> [!WARNING]
> This project is in an incredibly early state.

GUI tooling for browsing and basic editing of [Dark Engine](https://en.wikipedia.org/wiki/Dark_Engine) models. It is written in C# using the [Godot engine](https://godotengine.org/).

## AI Policy

This repo operates under a strict "No LLM/AI contributions" policy:

- No LLMs for issues
- No LLMs for PRs
- No LLMs for communication

Any contributor found to be in violation of this policy will have their contribution rejected and be banned from future contributions.
